Para o backend, instale as seguintes dependências:

npm install nodemon -D
npm run dev //agora, a cada cambio que temno arquivo index.js, o nodemon recarrega automaticamente.
npm install mongoose

Para o frontend, instale as seguintes dependências:
npx create-react-app frontend / yarn create-react-app frontend
npm install axios
npm install @material-ui/core  //(versão 4)
npm install react-icons
